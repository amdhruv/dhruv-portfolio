## ✨ Key Features

* 🎨 **Custom Text Selection Effect**
  When users select text, the highlight color changes dynamically, enhancing visual feedback and user experience.

* 🕸️ **Double Tap Web Shoot Animation**
  A unique interactive feature where double-tapping triggers a web-shoot effect, adding a fun and engaging UI element.

* ⚡ **Smooth & Responsive Design**
  Fully optimized for mobile, tablet, and desktop devices.

* 💡 **Modern UI/UX**
  Clean layout with attention to detail for a professional look and feel.

* 📊 **Portfolio Showcase**
  Displays projects related to data analysis, development, and more.
